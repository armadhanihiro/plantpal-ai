# PlantPal AI 🌱

PlantPal AI is a web application designed to help users care for plants through an AI-powered experience.

## Current Features

- React single-page/multi-page layout
- Plant image upload preview
- Plant health score UI
- Plant information card
- Question suggestions
- Recent question history with localStorage
- Dark mode
- Responsive navigation

## Tech Stack

- React
- Vite
- JavaScript
- Lucide React
