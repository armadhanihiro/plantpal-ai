# PlantPal AI 🌱

PlantPal AI is a web application designed to help users care for plants through an AI-powered experience.

## Current Features

- Upload plant images
- Preview uploaded images
- Ask plant-related questions
- Display plant guidance responses
- Responsive React UI

## Tech Stack

- React
- Vite
- JavaScript
- Lucide React
