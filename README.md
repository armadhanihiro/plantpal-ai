# 🌱 PlantPal AI

PlantPal AI is a modern AI-powered plant care assistant built with React, Vite, and Gemini AI.  
Users can upload plant images, ask questions about their plants, and receive intelligent AI-generated analysis and care recommendations.

---

## 🚀 Live Demo

[PlantPal AI](https://plantpal-ai-xi.vercel.app/)

---

## ✨ Features

- AI-powered plant analysis using Gemini AI
- Multimodal image understanding
- AI-generated plant information
- Smart plant care assistant
- Dark mode support
- Fully responsive mobile-friendly design
- Recent question history with localStorage
- AI-based plant health observations
- Prompt-engineered AI responses

---

## 🛠️ Tech Stack

- React
- Vite
- Gemini AI API
- React Markdown
- Vercel

## ⚙️ Installation

```bash
git clone https://github.com/your-username/plantpal-ai.git
```

```bash
cd frontend
npm install
npm run dev
```

Create `.env` file:

```env
VITE_GEMINI_API_KEY=your_api_key_here
```
